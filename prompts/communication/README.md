Communication Prompts

This folder contains prompts designed to assist with professional communication across platforms such as LinkedIn, email, internal company channels, and public project announcements.

The prompts included here help with:

Writing LinkedIn posts in a professional, humble tone

Crafting technical announcements or project summaries

Communicating with recruiters or hiring managers

Improving personal branding without exaggeration

Drafting clear and concise messages for a professional audience

Use this folder for any prompt related to public communication, professional messaging, branding, or networking.
