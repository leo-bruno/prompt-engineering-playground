# LinkedIn – Professional & Humble Post Generator

Generate a LinkedIn post with the following characteristics:

• Tone: professional, humble, and low-ego (no hype, no guru language).  
• Target audience: recruiters, technical peers, and people interested in practical AI applications.  
• Style: clear, concise, and authentic.  
• Message: I want to share a personal or technical project I’ve been working on, without overselling it.  
• Include:  
  – What the project is  
  – Why I built it / what I learned  
  – A link to the tool if available  
  – An optional note saying I might publish a related website in the future  
  – A closing sentence inviting friendly feedback (“Any feedback is more than welcome”).  
• Avoid: buzzwords, exaggerated claims, or marketing tone.  
• Write naturally, as if a QA engineer sharing a useful side project.

Structure to follow:
1. Short introduction to the project  
2. What problem it addresses or what I explored/learned  
3. Optional mention of a future website  
4. Link placeholder  
5. A closing feedback invitation  

Do not exceed 6–7 short lines.  
Write the final post in Spanish.
